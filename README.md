# BMI-Haptics

## 📦 Description
The BMI Haptics team is responsible for the developing the afferent neural pathway in a prosthesis: provide sensory feedback to the patient. Before integrating the technology in the prosthesis, a standalone haptic device is designed for reasearch and validation purpose.
